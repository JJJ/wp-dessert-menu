Grabs a WordPress Navigation Menu, and puts it in your Admin Bar

== Description ==

This little plugin has one simple purpose, to put a WordPress Nav Menu in your Admin Bar. It comes fully documented to allow you to reuse it for any number of custom menus.

== Installation ==

1. Place the 'wp-dessert-menu' folder in your '/wp-content/plugins/' directory.
2. Activate WP Dessert Menu.
3. Create a navigation menu in Admin > Appearance > Menus named 'Quick Links'
4. Sit back, relax, and enjoy .
5. To create custom menus, check out the fully documented code example in wp-dessert-menu.php.
